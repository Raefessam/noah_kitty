This folder is intentionally kept for future artwork (e.g. custom drawn sprites
or music files), but Noah's Kitty Adventure v1.0 needs no external assets at
all — every visual (Luna the kitten, trees, flowers, clouds, the flag, the
UI) is drawn live with Canvas code in renderer.js, and every sound is
generated live with the Web Audio API in audio.js. This keeps the game a
single self-contained folder that works completely offline.
